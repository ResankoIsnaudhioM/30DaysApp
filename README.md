# PRACTICE 30 Days Of Recipes App

Proyek ini merupakan latihan implementasi **UI Desain**, pengolahan **data resep**, serta penambahan elemen visual pada aplikasi mobile bertema resep masakan selama 30 hari.  
Aplikasi ini dibuat dengan tujuan untuk mempelajari dasar-dasar **Kotlin lanjutan** berdasarkan kursus Android Developer (Kotlin pada Android Studio).

## Fitur
- Menampilkan berbagai **resep makanan** selama 30 hari.
- Implementasi **Data Class**, **UI Desain**, **Card Layout**, dan **Typography**.
- Menampilkan detail resep seperti **bahan-bahan**, **waktu memasak**, dan **deskripsi singkat**.
- Navigasi sederhana antar halaman daftar dan detail resep.

## Tech
- **Android Studio** – IDE resmi untuk pengembangan aplikasi Android
- **Kotlin** – bahasa pemrograman modern untuk Android, ringkas & aman
- **XML** – untuk membuat layout antarmuka pengguna (jika tidak full Compose)
- **Android Jetpack** – kumpulan library pendukung pengembangan Android
- **Material Design Components** – komponen UI Material dari Google
- **Gradle** – sistem build otomatis untuk Android Studio
